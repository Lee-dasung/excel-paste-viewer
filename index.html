사용 방법

🔧 방법 1: 메모장 경유 (추천)

  1. 엑셀에서 데이터 선택 후 복사 (Ctrl+C)
  2. 메모장(notepad) 열고 붙여넣기 (Ctrl+V)
  3. 메모장 내용 전체 선택해서 복사
  4. 이 사이트 "텍스트 입력"에 붙여넣기

📁 방법 2: 파일 저장

  1. 엑셀에서 "다른 이름으로 저장"
  2. 파일 형식을 "CSV" 선택
  3. 저장된 파일을 "파일 업로드"에서 선택

⌨️ 방법 3: 직접 타이핑

  1. 엑셀 화면 보면서 직접 입력
  2. 열 구분은 탭, 콤마, 스페이스 사용
  3. 소량 데이터에 효과적

📋 지원하는 구분자

  • 탭: 열1[탭]열2[탭]열3
  • 콤마: 열1,열2,열3
  • 스페이스: 열1 열2 열3
  • 파이프: 열1|열2|열3
