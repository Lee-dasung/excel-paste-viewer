데이터 형식 변환

보안 PC에서 데이터 사용법

🔧 방법 1: 메모장 경유

  1. 엑셀에서 원하는 데이터 선택 → Ctrl+C
  2. 메모장(notepad) 열기 → Ctrl+V
  3. 메모장 내용 전체 선택 → Ctrl+C
  4. 이 사이트의 "수동 입력" 탭에서 Ctrl+V

📁 방법 2: 파일 저장 후 업로드

  1. 엑셀에서 "다른 이름으로 저장" → CSV 형식 선택
  2. 저장된 파일을 "파일 업로드" 탭에서 선택
  3. 자동으로 표 형식으로 변환됨

⚡ 방법 3: 스크린샷 + 수동 입력

  1. 엑셀 화면 캡처 (PrintScreen)
  2. 데이터를 보면서 직접 타이핑
  3. 소량의 데이터에 효과적

📋 지원되는 데이터 형식

  • 탭 구분: 열1\t열2\t열3
  • 콤마 구분: 열1,열2,열3
  • 공백 구분: 열1 열2 열3
  • 파이프 구분: 열1|열2|열3
